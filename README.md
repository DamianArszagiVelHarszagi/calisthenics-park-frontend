## Sources 🗃️
https://stackoverflow.com/questions/28371982/what-does-this-refer-to-in-arrow-functions-in-es6
= Used in map.js to debug an issue related to arrow functions. Because the function relied on this, I replaced the arrow function with a regular function so that this correctly referred to the expected context.

https://www.w3schools.com/jquery/default.asp
= Used as a reference for the jQuery library implemented in map.js.

https://stackoverflow.com/questions/8352372/how-to-add-load-more-button-for-a-html-css-page
= Used in browse.js to implement the “load more” functionality. I specifically targeted #areasList .area-card, which also helped me fix a related error.

https://mimo.org/glossary/javascript/settimeout
= Used in browse.js to implement a debounce effect for the filter input. setTimeout delays the filtering until the user stops typing for a short time, and clearTimeout cancels the previous timer to prevent the filter from running on every keystroke.

https://stackoverflow.com/questions/5353461/unicode-character-for-x-cancel-close
= Used in browse.js for the overlay close button. I used a Unicode “×” character because it looks cleaner than a standard letter "X".
